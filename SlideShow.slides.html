( Structure of Dataset )¶

The dataset comprised of borrower APRs and traits of 113,937 advances.Most variables are numeric and categorical. The credits included original loan amount. borrower's Prosper rating advance term. borrower's expressed month to month pay. just as numerous different elements like borrower's work status relationship of outstanding debt to take home pay. current advance status and so forth.¶

def ModifyData():
    df['IncomeRange'].replace(['Not employed', 'Not displayed'], '$0', inplace = True)
    ordinal_rating = ['$100,000+', '$75,000-99,999', '$50,000-74,999', '$25,000-49,999', '$1-24,999', '$0']
    ordered_var = pd.api.types.CategoricalDtype(ordered = True, categories = ordinal_rating)
    df['IncomeRange'] = df['IncomeRange'].astype(ordered_var)

    rate_order = ['HR','E','D','C','B','A','AA']
    ordered_var = pd.api.types.CategoricalDtype(ordered = True,categories = rate_order)
    df['ProsperRating (Alpha)'] = df['ProsperRating (Alpha)'].astype(ordered_var)

def BoxPlot(Data,figSize,X,Y,Title):
    plt.figure(figsize = figSize)
    base_color = sb.color_palette()[0]
